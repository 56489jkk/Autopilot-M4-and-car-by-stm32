1.绕X轴顺时针选转为roll角度的正方向    （横滚角）
1.绕Y轴顺时针选转为pitch角度的正方向  （俯仰角）         
1.绕Z轴顺时针选转为yaw角度的正方向       （偏航角）

飞控核心任务：3
1.imu传感器数据读取、校准、滤波，得到用于姿态解算和姿态控制的数据。
2.姿态解算、位置估计，得到实时可靠的角度、高度、速度等数据。
3.姿态控制，进行闭环控制飞行器，由于飞行器不是线性系统所以控制器要根据系统设计，才能使飞行器更加稳定。
4.遥控器数据读取、处理，把摇杆数据模型化，辅助通道用于飞行模式选择等。
